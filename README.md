# Personalized Webpage Project

A simple, user-friendly webpage that asks for your name and age, saves your information, and creates a personalized experience using JavaScript.

## Features Included

* **User Input & Storage**: Form collects your name and age and saves it using `localStorage` so it remembers you when you refresh.
* **Personalized Greeting**: Displays a welcome message using your name.
* **Age Calculation**: Automatically calculates and displays your age in total months.
* **Age Verification**: Checks if you are over 18 and displays a custom message based on your age.
* **Motivational Quotes**: Uses a JavaScript loop to print out a motivational quote 5 times.
* **Reset Button**: Easily clears saved data from your browser memory with one click.

## File Structure

* `index.html` - The structure of the website.
* `script.js` - The JavaScript logic for calculation, storage, and loops.
* `README.md` - Information about the project.

## How to Run Locally

1. Create a folder on your computer.
2. Save your `index.html` and `script.js` inside that folder.
3. Double-click the `index.html` file to open it in any web browser.

## How to Deploy to GitHub Pages

1. Upload your files to a public repository on GitHub.
2. Go to the repository **Settings**.
3. Click on **Pages** in the left menu.
4. Under "Build and deployment", set the source branch to **main** (or **master**) and root folder, then click **Save**.
5. Your site will be live at: https://github.com/Crispin804/personalized-webpage.git
